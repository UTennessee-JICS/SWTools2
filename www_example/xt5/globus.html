Category: Tools-Grid

Description

The Globus Toolkit is an open source software toolkit used for building Grid systems and applications. The tools in this package allow you to interact with other systems on the TeraGrid.

Use

These tools are typically for use on the login nodes only. They provide methods for obtaining grid certificates, passwordless SSH, and parallel file transfers. 

module load globus
myproxy-logon -l my_teragrid_username
gsissh kraken-pwd.nics.utk.edu

Support

This package has the following support level : Unsupported
