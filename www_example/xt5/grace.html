Grace

Category: Analysis-Visualization

Description

Grace is a WYSIWYG tool to make two-dimensional plots of data. Graphs can be produced in JPG, EPS, PNG, and several other formats.

Use

xmgrace
will launch the X11/Motif frontend. 
gracebat
can also be used perform rendering in batch mode. See the documentation at http://plasma-gate.weizmann.ac.il/Grace/doc/UsersGuide.html

Support

This package has the following support level : Unsupported
