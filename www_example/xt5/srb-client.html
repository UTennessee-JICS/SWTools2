Storage Resource Broker Client

Category: Tools-Data Management

Description

Storage Resource Broker (SRB) is a tools that supports shared data collections distributed across multiple organizations.

Use

SRB requires a bit of setup to get working. Once a collection is setup at an appropriate institution, one can use the 'S commands' to manipulate files. 

module load globus
grid-proxy-init
module load srb-client
Sinit
Sls

Support

This package has the following support level : Unsupported
