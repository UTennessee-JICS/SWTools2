Java Development Kit

Category: Program Dev-Languages

Description

The JDK is a development environment for building applications, applets, and components using the Java programming language.

More information can be found on the Java site.

Use

To add the Java Development Kit binaries, libraries, and man pages to your environment, load the java-jdk module. 

module load java-jdk
