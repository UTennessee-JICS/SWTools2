Mercurial

Category: Program Dev-Revision Control

Description

Mercurial is a fast and lightweight source control management system designed for efficient handling of very large distributed projects. Unlike tools such as CVS and Subversion, Mercurial is completely distributed and does not possess a centralized client/server architecture.

Use

To use Mercurial, one must first load the appropriate module: 

  module avail mercurial
  module load mercurial

The unofficial Mercurial manual, Distributed Revision Control with Mercurial gives a good description of Mercurial's use.

Support

This package has the following support level : Supported
