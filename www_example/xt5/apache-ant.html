Apache Ant

Category: Program Dev-Tools

Description

Apache Ant is a software tool for automating software build processes. It is similar to make but is implemented using the Java language, requires the Java platform, and is best suited to building Java projects.

Use

Apache-ant is available as a module, which sets environment variables pointing to the site-specific installation location. 

 module avail apache-ant
 module display apache-ant
 module load apache-ant

Support

This package has the following support level : Unsupported
