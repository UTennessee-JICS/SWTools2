UNICORE

Category: Tools-Grid

Description

UNICORE (Uniform Interface to Computing Resources) offers a ready-to-run Grid system including client and server software. UNICORE makes distributed computing and data resources available in a seamless and secure way in intranets and the internet.

Use

UNICORE includes both a command line client and X11-based graphical clients. To access them, one must first load the appropriate module: 

module avail unicore
module load unicore
Documentation for the command line client utility (ucc), the GPE client (application-client.sh), and the Eclipse-based rich client (UNICORE_Rich_Client) can be found in the UNICORE manuals.

Support

This package has the following support level : Unsupported
