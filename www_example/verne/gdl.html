GDL -- The GNU Data Language

Category: Analysis-Visualization

Description

GDL is a free Interactive Data Language (IDL) compatible incremental compiler and runtime environment. It is fully syntax compatible with IDL 6.0, with all IDL language elements supported. NetCDF, HDF, and HDF5 files are supported to varying degrees.

IDL is a registered trademark of ITT Visual Information Solutions.

Use

module swap PE-pgi PE-gnu
module load gdl
gdl
