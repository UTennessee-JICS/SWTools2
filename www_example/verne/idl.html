IDL

Category: Analysis-Visualization

Description

IDL is a software package for data analysis, visualization, and cross-platform application development. It combines tools for any type of project, from "quick-look" interactive analysis and display to large-scale commercial programming projects. It can be used to develop algorithms, interfaces, and visualizations and crunch through large numerical problems.

For more information, please see the ITT IDL site.

Use

On the analysis machines, IDL is available as a module 

  module load idl
  idl

Support

This package has the following support level : Supported
