PyNIO

Category: Libraries-Python-Graphics

Description

According to the "New users" section of the PyNGL website:

"PyNGL (pronounced "pingle") is a Python language module used to visualize scientific data, with an emphasis on high quality 2D visualizations."

Use

Versions of PyNGL are available as modules. Additional information about PyNGL is available on the PyNGL website. 

	module avail pyngl
	module load pyngl

Support

This package has the following support level : Unsupported
