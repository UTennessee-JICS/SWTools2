Python

Category: Program Dev-Languages

Description

Python is a clear and powerful object-oriented programming language, comparable to Perl, Ruby, Scheme, or Java.

For mor information see the Python home page .

Use

An older version of python (2.4.3) is available by default when you login. However, if you wish to use a newer python, then you need to load a python module. 
 
  module avail python
  module load python/2.5.2
