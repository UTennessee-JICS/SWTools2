CDO

Category: Analysis-Mathematics

Description

The Climate Data Operators software is a collection of operators for standard processing of climate and forecast model output. The operators include simple statistical and arithmetic functions, data selection and subsampling tools and spatial interpolation. CDO was developed to have the same set of proceessing functions for GRIB and netCDF datasets in one package. It is used for fast, format independent access to GRIB and netCDF datasets.

Use

See CDO web page for more information.

Load the cdo module: 

module avail cdo
module load cdo
