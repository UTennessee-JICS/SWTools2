AZTEC

Category: Libraries-Math-Parallel

Description

Trilinos is a suite of parallel object-oriented software framework for the solution of large-scale, complex multi-physics engineering and scientific problems. It is a product of Sandia National Laboratory. For detail information, please visit http://trilinos.sandia.gov.

Use

Load the Trilinos module. 

  module avail trilinos
  module load trilinos/8.0.3
  cc test.c ${TRILINOS_PATH} -l[the needed libraries in a correct order]
Note that if you change the programming environment after loading the aztec module, you will have to reload or update the aztec module. Please refer to examples in ${TRILINOS_DIR}/examples/aztecoo/UG_Ex1 for additional information.
