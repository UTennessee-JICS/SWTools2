Ruby

Category: Program Dev-Languages

Description

A dynamic, open source programming language with a focus on simplicity and productivity. It has an elegant syntax that is natural to read and easy to write.

For more information see the Ruby home page .

Use

An older version of ruby (1.8.4) is available by default when you login. However, if you wish to use a newer ruby, then you need to load a ruby module. 
 
  module avail ruby 
  module load ruby/1.9.1
