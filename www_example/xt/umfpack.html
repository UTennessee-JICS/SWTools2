UMFPACK

Category: Libraries-Math

Description

A set of routines solving sparse linear systems via LU factorization. Requires three other packages: the BLAS (dense matrix operations), AMD (sparse matrix minimum degree ordering), and UFconfig. Includes a C-callable and MATLAB interface, and a basic FORTRAN 77 interface to a subset of the C-callable routines. Requires AMD Version 2.0 or later.

Use

This library is only built for use on the service nodes. 
module load umfpack
cc -o example example.c ${UMFPACK}

Support

This package has the following support level : Unsupported
