Ferret

Category: Analysis-Visualization

Description

Ferret is an interactive computer visualization and analysis environment designed to meet the needs of oceanographers and meteorologists analyzing large and complex gridded data sets.

Use

For documentation and instructions, see the NOAA website at http://ferret.pmel.noaa.gov. 

module load ferret
ferret

Known Issues:

  • This version of Ferret is statically linked and thus does not support loadable external functions.
  • The -gui option does not render correctly and may freeze your environment.
  • You may experience an X-Window problem where it does not like your DISPLAY setting. Take the number ## in your DISPLAY setting hopefully inherited from X-forwarding which should look like "localhost:##.0" and then set DISPLAY to 127.0.0.1:##.0.

Support

This package has the following support level : Unsupported
