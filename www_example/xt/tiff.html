TIFF

Category: Analysis-Graphics

Description

The Tag Image File Format (TIFF) is a widely used format for storing image data.

Included in this software distribution is a library, libtiff, for reading and writing TIFF, a small collection of tools for doing simple manipulations of TIFF images, and documentation on the library and tools. Libtiff is a portable software, it was built and tested on various systems: UNIX flavors (Linux, BSD, Solaris, MacOS X), Windows, OpenVMS.

Use

 module avail tiff
 module display tiff
 module load tiff
