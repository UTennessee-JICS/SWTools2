Pacman

Category: Tools-Package Management

Description

Pacman is a simple library-based package manager. It uses simple compressed files as a package format and maintains a text-based database of packages.

Use

	module load pacman
	pacman -install <url to packman package>

Support

This package has the following support level : Unsupported
