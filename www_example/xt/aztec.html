AZTEC

Category: Libraries-Math-Sparse

Description

Aztec is a standalone parallel iterative solver for large systems of sparse linear equations. It is a standalone package developed by Sandia National Laboratory in the mid 90s. The library is written in C and MPI and is callable from Fortran. Aztec is now superseded by the Trilinos solver framework, http://trilinos.sandia.gov. You can also refer to the user manual available in /sw/sources/aztec/doc for additional information.

Use

Load the aztec module. 

  module avail aztec
  module load aztec/2.1
  cc test.c ${AZTEC_LIB}
Note that if you change the programming environment after loading the aztec module, you will have to reload or update the aztec module.

Support

This package has the following support level : Unsupported
