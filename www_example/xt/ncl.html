NCAR Command Language (NCL)

Category: Analysis-Visualization

Description

From "http://www.ncl.ucar.edu/overview.shtml":

The NCAR Command Language (NCL), a product of the Computational & Information Systems Laboratory at the National Center for Atmospheric Research (NCAR), is a free interpreted language designed specifically for scientific data processing and visualization.

As of version 5.0.0, NCL and NCAR Graphics are released as one package in source code or binary format.

Use

Load the module to gain access to the NCL binaries and NCAR Graphics libraries. 

module load ncl

See "http://www.ncl.ucar.edu/overview.shtml" for usage information.

The following shows what the ncl module sets (the actual path may change as new versions are installed.) 

> module display ncl
-------------------------------------------------------------------
/sw/xt5/modulefiles/ncl/5.0.0:

module-whatis    Sets up environment to use NCL and NCAR Graphics 5.0.0. 
setenv           NCARG_ROOT /sw/xt5/ncl/5.0.0/sles10.1_binary 
prepend-path     PATH /sw/xt5/ncl/5.0.0/sles10.1_binary/bin 
-------------------------------------------------------------------

If you wish to run an example, you can use the ng4ex command to generate an example. For example, to generate the gsun01n example, do 

ng4ex gsun01n
ncl gsun01n.ncl
